# UX Flows – WithYou

## Auth Flow
1. Welcome
2. Register / Login
3. Route based on relationship status

---

## Unpaired Flow
- Unpaired Home
  - Generate invite code
  - Enter invite code
  - Clear explanation of consent

---

## Pairing Flow
- Invite generation (auth required)
- Invite acceptance (auth required)
- Server enforces one active relationship per user
- Confirmation → Paired state

---

## Paired Tabs
- Dashboard
- Check-In
- Ideas
- Preferences
- Settings

---

## Check-In Flow
- Select mood (1–5)
- Optional note
- Optional guided reflection prompt
- Share toggle (off by default)
- Submit → Dashboard

---

## Ideas Flow
- Toggle mode:
  - Reconnect (default)
  - Repair
- Generate exactly 3 ideas
- Refresh allowed

---

## Weekly Reflection (Stage 2)
- Triggered once per week
- Private by default
- Mutual share required for visibility

---

## End Pairing
- Confirmation required
- Relationship state reset to Unpaired
